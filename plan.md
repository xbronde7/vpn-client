# План: Мобильное VPN-приложение (Hiddify/v2rayNG-like)

## Цель
Создать кроссплатформенное мобильное приложение для Android и iOS с функциональностью VPN-клиента, аналогичного Hiddify/v2rayNG — с поддержкой протоколов v2ray/Xray (VMess, VLESS, Shadowsocks, Trojan) и удобным интерфейсом управления подключениями.

## Стек
- **Фреймворк**: React Native (Expo) — для кроссплатформенной разработки
- **Сетевая часть**: Использование нативных модулей / bridge к существующим библиотекам (v2ray-lib, xray-core)
- **UI**: React Native + React Navigation + стилизация
- **Хранение**: AsyncStorage для конфигураций
- **QR/Сканирование**: expo-camera / expo-barcode-scanner для импорта конфигураций

## Этапы

### Stage 1 — Проектирование и дизайн
- Определение функциональных требований
- Проектирование архитектуры приложения
- Дизайн UI/UX (экраны, навигация, компоненты)

### Stage 2 — Исследование интеграции с VPN/прокси

- Анализ форматов конфигураций (JSON VMess/VLESS/SS/Trojan, ссылки, QR-коды)

### Stage 3 — Разработка ядра приложения
- Настройка проекта React Native (Expo)
- Реализация экранов UI
- Реализация логики управления конфигурациями
- Интеграция с VPN/прокси движком

### Stage 4 — Сборка и деплой
- Сборка APK (Android)
- Инструкции по сборке для iOS
- Деплой веб-версии (опционально, для демонстрации UI)
